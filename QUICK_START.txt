📋 EASIEST WAY TO RUN:
─────────────────────────────────────────────────────────────────

1. Open PowerShell in the project directory (d:\Qader\app-main)

2. Run the startup script:
   .\start-all.ps1

3. This will automatically start:
   ✅ MongoDB
   ✅ Backend API Server

─────────────────────────────────────────────────────────────────

🧪 TO RUN TESTS:
─────────────────────────────────────────────────────────────────

Option 1 (Using script):
   .\run-tests.ps1

Option 2 (Manual):
   d:\Qader\app-main\.venv\Scripts\python.exe backend_test.py

─────────────────────────────────────────────────────────────────

🌐 SERVICE URLs:
─────────────────────────────────────────────────────────────────

MongoDB:        localhost:27017
Backend API:    http://localhost:8000/api
Frontend:       http://localhost:3000 (when started)
